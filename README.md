# Daily English Speaking PWA

하루 10분 영어 회화 쉐도잉 및 스피킹 트레이닝을 위한 모바일 PWA 웹 애플리케이션입니다.

## 주요 기능
- 10일간의 일상 영어 대화 쉐도잉 코스 탑재
- Web Audio API 기반의 실시간 음성 시각화 (Waveform UI)
- webkitSpeechRecognition API 기반 영어 음성 인식 및 스마트 채점 (Levenshtein 알고리즘)
- Web Audio API 기반 효과음 신시사이저 탑재
- iOS 사파리 홈 화면 추가(PWA standalone) 최적화 및 100dvh 하단 안전영역 레이아웃 지원
